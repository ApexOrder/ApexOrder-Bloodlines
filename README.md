# ApexOrder Bloodlines

Standalone 7 Days to Die overhaul by ApexOrder.

## Current milestone

The first development milestone implements the Bloodlines introductory survival protocols:

1. Gather fibres, sticks and stones.
2. Craft fibre rope.
3. Craft a crude axe.
4. Harvest primitive materials, craft a crude pike and bark armour, then equip the armour.
5. Establish shelter with a bedroll and campfire.
6. Eliminate five infected.
7. Complete the Bloodlines transmission and continue into the vanilla White River quest.

## Repository layout

The installable mod is stored in `ApexOrder_Bloodlines/`. Release builds package that directory without modifying unrelated mods.

## Development status

This is an early development build. XML must be tested against the target 7 Days to Die version before publishing a stable release.
